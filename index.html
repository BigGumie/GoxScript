Introdução

No meu projeto para a PAP decidi criar um assistente de voz em python. Este relatório trata de explicar como foi o processo de desenvolvimento do mesmo, irei mostrar como funciona e como foi o método de criação do mesmo.

Porque Python?

A razão pela qual escolhi ir com Python foi para aprender. Não é uma linguagem que esteja no programa escolar mas é uma linguagem bastante usada no mundo atual por isso decidi sair da minha zona de conforto e fazer a minha PAP 100% a base de python.

Porque Hal?

Inicialmente o nome do projeto era Lucky, assim como “Siri” e “Alexa”, Lucky era um nome unisexo e foi essa a única razão pela qual o escolhi. Em fevereiro, comecei a achar que Lucky não era um nome real por isso decidi mudar para Ray, que embora não sendo unisexo, era curto e dava para fazer o slogan que tinha em mente, Responsive Assistant Yet another Mais para a reta final, achei que o slogan não fazia muito sentido, por isso decidi usar um nome igualmente curto e sem slogan mas referente ao filme “2001 - Odisseia no Espaço” que é o nome que decidi usar no final.

O que é um raspberry Pi?

Raspberry Pi é uma espécie de mini computador, criado para que toda a gente que estivesse interessada em aprender um pouco mais acerca de computação. Por esta razão, o Raspberry Pi é usado para vários projetos, sendo o meu um deles.
